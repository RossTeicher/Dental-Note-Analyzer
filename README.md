# AI Dental Note Generator (GPT-4 Vision)

This app generates detailed dental notes, diagnoses, and treatment options from uploaded procedure codes, perio charts, and radiographs using GPT-4 Vision.

## How to Run
1. Set your `OPENAI_API_KEY`
2. Install dependencies: `pip install -r requirements.txt`
3. Add system packages from `packages.txt`
4. Run with: `streamlit run app.py`
